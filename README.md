<h2>🛠️ Installation Steps:</h2>

<p>1. to Start</p>

```
npm run dev
```

<p>2. Create .env in root</p>

```
DATABASE_URL="YOUR DATABASE KEY"
```

<p>3. Also create .env.local</p>

```
NEXT_PUBLIC_BASE_URL="http://localhost:3000"
```

```
NEXTAUTH_SECRET="Your auth key"
```
